安裝 kubectl

gcloud components install kubectl

找出名稱特別長的

kubectl get rc

dcard-web-dskfodihvdslknfsdf 49 29 0d

砍掉它

kubectl delete rc/dcard-web-dskfodihvdslknfsdf

如果還是不行，要先 scale down，然後 rebuild:

kubectl scale  --replicas=20 rc/dcard-web
